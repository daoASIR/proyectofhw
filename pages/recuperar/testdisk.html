Test Disk

Test Disk es un potente programa de recuperación de datos para usuarios avanzados. Es capaz de recuperar particiones borradas o corruptas y los ficheros que contenian.

Es fácil de utilizar, aunque hay que tener cuidado al usarla ya que si se escoge la opción erronea se puede corromper toda la estructura del disco. Personalmente, me ha servido para recuperar la información de dos discos duros con particiones irreconocibles por el SO.

Tiene licencia GPLv2+ (Open Source) y puede recuperar particiones para los siguientes sistemas de archivos:

  • BeFS
  • BSD disklabel
  • CramFS
  • FAT12, FAT16 y FAT32
  • exFAT
  • HFS y HFS+
  • JFS
  • ext2, ext3 y ext4
  • Linux Raid
  • Linux Swap
  • LVM y LVM2
  • Mapa de particiones Mac
  • NSS
  • NTFS
  • ReiserFS 3.5, 3.6 y 4
  • i386 disklabel
  • UFS y UFS2
  • XFS

Windows

Portable

Test Disk se distruibuye en forma portable. Esto quiere decir que se puede usar sin necesidad de instalarlo. Si queremos tenerlo en el equipo como si fuera una versión instlada simplemente tenemos que descomprimir los ficheros en un lugar de nuestra elección (en mi caso he creado una carpeta «programas portables» en C:) y guardar un acceso directo al ejecutable en la siguiente ruta:

C:\Users\[$USUARIO]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs

De esta forma Windows lo añade ai índice del menú de inicio y permitiendo buscarlo desde allí.

Ubuntu

Terminal

La instalación en Ubuntu es muy sencila. Primero es recomendable actualizar las listas del repositorio:

sudo apt update

A continuación lo instalamos con el siguiente comando:

sudo apt install testdisk

Por último podemos comprobar que se ha instalado correctamente:

sudo dpkg -l testdisk

Para ejecutar el programa simplemente escribimos su nombre precedido por sudo:

sudo testdisk
