Blender

Blender es un programa multiplataforma dedicado principalmente al modelado tridimensional, aunque es capaz de hacer otras muchas cosas como composición digital mediante nodos, edición de vídeo, escultura y pintura digital.

Se trata de un programa gratuito y con licencia Open Source GPL3. Fue lanzado en 1994, y en los últimos años ha explotado su popularidad gracias a la versión 2.80. Entre sus características principales se encuentran:

  • Renderizado
  • Modelado 3D
  • Escultura
  • Animación
  • Efectos especiales
  • Simulaciones
  • Integración con herramientas del sector audovisual
  • Edición de vídeo
  • API de Python para uso de scripts

Sus requisitos mínimos son:

CPU 64bit x4 núcleos con soprtet SSE2
RAM 8GB
Pantalla 1920x1080
GPU 2GB VRAM, OpenGL 4.3

Se puede descargar de forma gratuita desde la web del desarrollador:

blender.org

Interfaz Gráfica

La interfaz de Blender ha cambión radicalmente en su versión 2.80, convirtiéndolo en uno de los programas Open Source para diseño 3D más populares. La interfaz es completamente modular y se basa en paneles, los cuales pueden mostrar cualquier tipo de información y colocarse en cualquier parte de la ventana.

En la galería se muestran varias configuraciones de los espacios de trabajo como el de edición 3D o el de edición de vídeo. Además, se muestra cada una de las secciones de la ventana de configuración.

Windows

Instalador

Para instalar Blender, se descarga el instalador desde la página del desarrollador y se ejecuta. Se sigue el asistente de instalación aceptando las opciones por defecto y se espera que finalice la instalación.

Portable

Instalar la versión portable de Blender para Windows se hace igual que la mayoría de versiones portables de otros programas. Se descarga un fichero comprimido y se descomprime el contenido en una carpeta, ya sea en el disco interno de nuestro ordenador o en uno externo para poder ejecutarlo en otros ordenadores.

Si queremos tenerlo en el equipo como si fuera una versión instalada simplemente tenemos que descomprimir los ficheros en un lugar de nuestra elección (en mi caso he creado una carpeta «programas portables» en C:) y guardar un acceso directo al ejecutable en la siguiente ruta:

C:\Users\[$USUARIO]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs

De esta forma Windows lo añade la índice del menú de inicio permitiendo abrir el programa desde allí.

Tienda

Otra opción para instalar Blender es hacerlo desde la tienda de Windows. Una ventaja de hacerlo desde aquí es que el programa se actualiza sin intervención del usuario.

Ubuntu

Terminal

Para instalar Blender desde la terminal primero es recomendable actualizar los repositorios:

sudo apt update

Se instala con el siguiente comando:

sudo apt-get install blender

Hay que destacar que la versión instalada con este método no es la última.

Portable

Otra opción, que además nos permite usar Blender en su versión más actualizada, es descargar el archivo portable desde la web del desarrollador. Para ello se descarga el fichero y se descomprime en la ubicación que deseemos. Una vez descomprimidos los archivo se puede borrar el fichero original con el fin de ahorrar espacio en el disco.

Para usar el programa, simplemente abrimos el fichero llamado blender que se encuentra dentro de la carpeta recién creada.
