Proyecto FHW & Lenguaje de Marcas

En esta web se detalla el proceso de instalación en Windows y Ubuntu de varios programas, a través de vídeos un minuto de duración. Además, en cada sección se podrá encontrar una breve descripción del programa, una galería de imágenes mostrando la interfaz con sus menús de configuración y una explicación en texto para cada método de instalación.

En las secciones principales se describe para qué sirve cada tipo de programa y se hace una comparative entre los programas que se han instalado.

Se han instalado 10 programas en distintas categorías, sumando en total 25 tutoriales de instalación y 2 demostraciones de uso. Los programas instalados han sido los siguientes:

  • Microsoft Office 365
  • LibreOffice 7.2
  • WPS Office
  • TestDisk
  • Recuva
  • Avast
  • eSeT
  • ClamAV
  • Krita
  • Blender

Por la parte del diseño de la web, se ha puesto en práctica lo siguiente:

  • Uso de JavaScrip con la ayuda de la librería jQuery
  • Carga dinámica de código HTML mediante JavaScript
  • Posicionamiento dinámico (con JS) de elementos en función del tamaño de ventana y posición del scroll
  • Uso de varias hojas de estilo, con tema claro y oscuro
  • Uso de CSS Avanzado
  • Uso de flexbox para ayudar a posicionar los elementos
  • Uso de etiquetas semánticas como header, nav, section, article, footer, etc
  • Diseño de menús de navegación
  • Uso de tablas y listas
  • Uso de etiquetas de imagen y vídeo
  • Validación de código HTML en W3C
  • Uso de CSS «responsive» para adaptar la web a:
    • Pantallas grandes
    • Pantallas pequeñas
    • Pantallas muy pequeñas
    • Impresión (Se ocultan los menús de navegación)
