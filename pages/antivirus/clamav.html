ClamAV

Clam Antivirus (ClamAV) es un programa Antivirus de línea de comandos gratuito y Open Source para sistemas Linux. Es usado para detectar troyanos y software malicioso, a pesar de que los sistemas Linux son atacados con mucha menos frecuencia que otros sistemas operativos.

Está disponible para las siguientes distribuciones de Linux:

  • Ubuntu
  • Debian
  • CentOS

Se puede descargar desde la web oficial del desarrollador de forma gratuita o desde la terminal de Linux.

ClamAV

Interfaz Gráfica

ClamAV es en realidad un programa de línea de comandos. Opcionalmente se le puede instalar una GUI llamada ClamTK. Presenta una interfaz simple con las opciones básicas de cualquier antivirus como actualizar la base de datos de virus, analizar un directorio y los archivos en cuarentena.

Ubuntu

Terminal

Para instalarlo desde la terminal primero hay que actualizar los repositorios del sistema:

sudo apt-get update

El programa se instala con el siguiente comando:

sudo apt-get install clamav clamav-daemon

Si se desea se puede comprobar que versión se ha instalado:

clamscan --version

Por último, se le puede añadir una GUI instalando ClamTK:

sudo apt-get install clamtk
