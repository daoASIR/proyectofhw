Microsoft Office 365

Microsoft Office 365 es una suite ofimática de código cerrado desarrollada por Microsoft. Su modelo de venta es el de la suscripción anual, con un precio 99€/año en su versión familiar para 5 usuarios. Incluye los siguientes programas:

Word Procesador de texto
Excel Hoja de cálculo
PowerPoint Presentaciones
Publisher Maquetación
Access Gestor de bases de datos
Outlook Cliente de correo electrónico
One Note Programa de notas

Está disponible para los principales sistemas operativos de escritorio, asi como dispositivos móviles y versión web. Se puede descargar desde la web oficial de Microsoft previo pago de la suscripción. El enlace de descarga se encuentra en la página de gestión de suscripciones.

Office.com

Interfaz Gráfica

MS Office tiene una interfaz limpia e intuitiva, lo que hace que trabajar con ella sea sencillo incluso para las personas con menos experiencia.

Utiliza una interfaz tipo cinta que agrupa los botones por usos, facilitando el acceso a las opciones que se necesitan en cada momento.

Windows

Ejecutable

Instalar MS Office es sencillo una vez se ha adquirido la licencia. Simplemente hay que iniciar sesión en la web de Microsoft y descargar la suite. La instalación comenzará automáticamente.

Si no tenemos la cuenta de Windows vinculada a nuestra cuenta de Microsoft, es posible que se nos pida iniciar sesión al abrir la suite por primera vez. Además, en el primer inicio nos pedirá que configuremos una serie de opciones sobre privacidad y formato de archivos predeterminados.

Tienda

En caso de tener la cuenta de Microsoft vinculada a la de Windows, la Tienda de Windows es otra opción bastante sencilla de instalar Office. Simplemente hay que buscarlo en la aplicación de la tienda y darle a instalar (o comprar en caso de que aún no se disponga de la licencia).

Navegador

Version web

Aunque más limitada en funcionalidad, la versión web de Microsoft Office es gratuita y se puede ejecutar desde cualquier navegador. Esto permite que se pueda utilizar independientemente del sistema operativo que se tenga en la máquina.
