WPS Office

WPS Office es una suite ofimática disponible para los sistemas operativos Windows, Linux, Android e iOS creada por la empresa china Kingsoft Office. La versión básica es gratuita, aunque se trata de software propietario. Su interfaz es similar a la de Microsoft Office y es compatible con los formatos propietarios de éste.

Windows

Instalador

Instalar WPS Office en Windows no supone ninguna dificultad. Simplemente lo descargamos desde la página del desarrollador y lo instalamos ejecutando el fichero. Se nos pedirá que aceptemos los términos de uso y le damos a instalar. El único inconveniente que puede causar es que por defecto crea dos iconos en el escritorio y se ancla a la barra de tareas sin pedir permiso, aunque tiene fácil solución.

Tienda

Una forma todavía más fácil es instalarlo desde la tienda de Windows. Para ello simplemente abrimos la tienda y buscamos el programa. Una vez en la ficha de WPS Office, le damos al botón que pone «instalar».

Ubuntu

Terminal

En sistemas Linux es recomendable actualizar las listas del repositorio antes de instalar nuevo software. El comando para hacerlo es:

apt update

Podemos descargar el paquete .deb desde la web del desarrollador o pasarle el enlace al siguiente comando para que los descargue desde la terminal:

wget https://wdl1.pcfg.cache.wpscdn.com/wpsdl/wpsoffice/download/ linux/10702/wps-office_11.1.0.10702.XA_amd64.deb -P ~/Downloads

Por último, instalamos el software que acabamos de descargar con:

sudo apt install ~/Downloads/wps-office_11.1.0.10702.XA_ amd64.deb
