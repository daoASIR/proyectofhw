LibreOffice 7.2

LibreOffice es una suite ofimática gratuíta y de código abierto creada en 2010 como una bifurcación de OpenOffice tras la adquisición de ésta por parte de Oracle. Cuenta con un procesador de texto, un editor de hojas de cálculo, un gestor de presentaciones, un gestor de bases de datos, un editor de gráficos vectoriales y un editor de fórmulas matemáticas.

Writer Procesador de texto
Calc Editor de hojas de cálculo
Impress Gestor de presentaciones
Math Editor de notación matemática
Base Gestor de bases de datos
Draw Editor de gráficos vectoriales

Se puede descargar completamente gratis desde la página web oficial del proyecto LibreOffice.

Windows

Ejecutable

Se descarga el instalador desde la página oficial del desarrollador y se ejecuta. Se ha realizado una instalación con las opciones por defecto y sin crear icono en el escritorio. Un vez finalizada la instalación se podrán abrir los programas escribiendo su nombre en el menú de inicio.

Portable

La versión portable de libreOffice se puede descagar desde el menú «descargas» en la parte superior de la web oficial. Ejecutamos el fichero y se abre un asistente de instalación que pregunta dónde se quieres guardar los ficheros del programa y si se desean descartar los idiomas adicionales con el fin de ahorrar espacio.

Una vez finalizada la copia de los ficheros, podemos guardar la carpeta en un dispositivo externo y ejecutar libreOffice desde cualquier equipo con Windows donde lo conectemos.

Ubuntu

Terminal

Para instalar LibreOffice en linux desde la terminal primero hay que añadir el repositorio desde el que descargarlo con el siguiente comando:

sudo add-apt-repository ppa:libreoffice

Una vez añadido el repositorio, se instala con el siguiente comando:

sudo apt-get install libreoffice

Tienda

Otra opción es instalar LibreOffice desde la tienda integrada en el sistema operativo. En unbuntu se llama «Ubuntu Software» y permite instalar las aplicaciones de LibreOffice de forma individual.
