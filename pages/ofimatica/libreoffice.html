LibreOffice 7.2

LibreOffice es una suite ofimática gratuita y de código abierto. Fue creada en 2010 como una bifurcación de OpenOffice tras la adquisición de ésta por parte de Oracle. Cuenta con un procesador de texto, un editor de hojas de cálculo, un gestor de presentaciones, un gestor de bases de datos, un editor de gráficos vectoriales y un editor de fórmulas matemáticas.

Writer Procesador de texto
Calc Editor de hojas de cálculo
Impress Gestor de presentaciones
Math Editor de notación matemática
Base Gestor de bases de datos
Draw Editor de gráficos vectoriales

Se puede descargar gratis desde la página web oficial del proyecto LibreOffice.

LibreOffice.org

Interfaz Gráfica

LibreOffice tiene una interfaz organizada por menús en el que la mayoría de las opciones están disponibles a un click de distancia. Aunque es menos eficiente en el uso del espacio disponible en pantalla y menos intuitiva que otras opciones de la competencia, sigue siendo una buena opción una vez superada la curva de aprendizaje.

Windows

Instalador

Se ha realizado una instalación con las opciones por defecto y sin crear icono en el escritorio. Una vez finalizada la instalación se podrán abrir los programas escribiendo su nombre en el menú de inicio.

Portable

La versión portable de libreOffice se puede descargar desde el menú «descargas» en la parte superior de la web oficial. Tras ejecutar el fichero se abre un asistente de instalación que pregunta dónde se quieren guardar los ficheros del programa. Además, da la opción de descartar los idiomas adicionales con el fin de ahorrar espacio.

Una vez finalizada la copia de los ficheros, podemos guardar la carpeta en un dispositivo externo y ejecutar libreOffice desde cualquier equipo al que lo conectemos (y que tenga Windows). Funcionalmente el programa es el mismo que su versión instalable, con la ventaja de poder utilizarlo en varios equipos conservando las configuraciones que hagamos.

Ubuntu

Terminal

Para instalar LibreOffice en linux desde la terminal, primero hay que añadir el repositorio desde el que descargarlo con el siguiente comando:

sudo add-apt-repository ppa:libreoffice

Una vez añadido el repositorio, se instala con el siguiente comando:

sudo apt-get install libreoffice

Tienda

Otra opción es instalar LibreOffice desde la tienda integrada en el sistema operativo. En Ubuntu se llama «Ubuntu Software» y permite instalar las aplicaciones de LibreOffice de forma individual.
